 ## Usage of the program 

### create a virtual environment and install the requirements
for mac/linux
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
for windows
```bash
python3 -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### run the program
```bash
python3 weather.py
```
